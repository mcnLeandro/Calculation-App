# My-iPhone's-Calculation-App
自分が使っているiPhoneに入ってる計算機をWebApplicationにしました。
僕のiPhoneの計算機を使うことができます。

ふと、iPhoneの計算機が使いたい、でもiPhoneは手元にない、ブラウザは開いてる、という場面で問題なく、iPhoneのCalculation-Appが使えます。

ブックマークをしておくと便利です。
